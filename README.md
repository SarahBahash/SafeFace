# SafeFace – Real-Time Micro-Expression Analysis System

SafeFace is an AI-driven system that analyzes facial micro-expressions in real time to detect abnormal stress levels and potential security risks.  
The solution is designed for government service environments and prioritizes user privacy by avoiding any image storage.

---

## Key Features
- Real-time facial expression and stress analysis  
- Automatic risk-level classification (Low / Medium / High)  
- Face tracking with auto-pause when the subject leaves the frame  
- Multi-camera monitoring dashboard  
- Smart alerting and activity logging  
- Fully Arabic, government-style interface
